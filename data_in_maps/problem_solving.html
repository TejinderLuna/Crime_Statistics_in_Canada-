Problem 1

Firefox can pass Google OAUTH, but Chrome.

Symptom:

Firefox can pass Google OAUTH successfully, but Chrome. No error log in the console of either Firefox or Chrome.

Problem solving:

I added many checkpoints in the script in order to find out the breakpoint.

The auth code example downloaded from Google YouTube API website is problematical. It doesn't check the authentication result error code. If browser does not have account.google.com cookie, the authentication will be failed. But browser will not display any error since authentication failure is one of the normal result as well. Web programmer should handle it, not browser.

Problem 2

Circle radius cannot be controlled.

Symptom:

The radius of circle on the map is based on crime volume. Sometimes, the large is very large, the small is very small.

Problem solving:

There are many scale formulas on the web. But none of them works fine in my project.

Scale the crime volume value to a range from 0 to 100. Then I am able to control the radius of circles.

Problem 3

Integer array is not sorted properly.

Symptom:

Integer array cannot be sorted properly even if each element has been parsed to integer.

Problem solving:

The numbers are sorted as strings even if they are parsed into integer.

I added a customized function in JQuery built-in sort() method.

  • Return greater than 0 if a is greater than b
  • Return 0 if a equals b
  • Return less than 0 if a is less than b

When I call sort(), the customized function is called on pairs in to-be-sorted list, to determine the proper ordering.
